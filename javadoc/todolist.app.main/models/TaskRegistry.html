public class TaskRegistry
extends java.util.ArrayList<Task>
implements java.io.Serializable
Persistent, ordered list of Task objects.

Provides methods for filtering and reordering tasks. Upon creation, it loads tasks from a file if it exists, else it creates a new file. Tasks are written to or removed from the file as they are added to or removed from the list.
