    • plural

      public static java.lang.String plural​(java.lang.String str, int n)
      Pluralise a word.

      Takes a String and an int. Returns a String consisting of the provided int, a space, the provided String with an "s" appended if the int is not equal to 1, and a space.

      Example: plural("apple", 1) will return "1 apple ", plural("green onion", 2) will return "2 green onions ". It does not handle irregular words; plural("sheep", 3) returns "3 sheeps" (sic).

      Parameters:
      str - word or sentence to pluralise
      n - number of occurrences
      Returns:
      pluralised string ending with a space
